# 町内会計管理アプリ - CLAUDE.md

## プロジェクト概要

26世帯の町内会を対象とした会計管理Webアプリ。
従来Excelで行っていた収支管理・集計・総会報告をブラウザ上で完結させる。

## 開発者情報

- 開発者：初心者（Claude Codeで学習中）
- 目的：HTML/CSS/JavaScriptの習得 + 実用ツールの作成

## 技術スタック

- HTML / CSS / JavaScript（フロントエンドのみ、サーバー不要）
- データ保存：LocalStorage
- 出力：CSV（Excel対応）
- バージョン管理：GitHub

## ファイル構成

```
町内会計管理アプリ/

├── style.css    # デザイン
├── app.js       # ロジック・データ処理

```

## コーディングルール

- コードの各行または各ブロックに**日本語コメント**を必ず入れる
- 変数名・関数名は英語、コメントは日本語
- 初心者が読んでわかる丁寧なコメントにする
- 1つの関数は1つの役割のみ（シンプルに保つ）

## データ仕様

### 1件あたりのデータ構造

```javascript
{
  id: "一意のID（自動生成）",
  date: "2025-01-15",       // 日付
  type: "income",           // "income"（収入）or "expense"（支出）
  category: "町内会費",      // カテゴリ名
  amount: 5000,             // 金額（円）
  note: "1月分"             // 摘要（任意）
}
```

### 収入カテゴリ

- 町内会費
- 補助金
- 繰越金
- その他収入

### 支出カテゴリ

- 行事費
- 消耗品費
- 通信費
- 慶弔費
- その他支出

## Claude Codeへの指示方針

- 新機能追加時：「〇〇機能を追加してください。各行に日本語コメントを入れてください。」
- エラー発生時：エラーメッセージをそのまま貼り付けて修正依頼
- 理解したい時：「この部分を初心者向けに説明してください。」
- 一度に大きな変更をせず、小さく作って動作確認してから次へ進む
